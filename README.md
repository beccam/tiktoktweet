## Requirements

The current version of TikTokTweets works with:
* Django (see the docs)
* cqlengine (Cassandra CQL 3 Object Mapper for Python)
* Tweepy
* Django Cassandra Engine 0.2.2 (the Cassandra backend for Django)
* Cassandra 2.x
* Python 2.7
